Enzymes makes it extremely easy to move (and execute !?) custom fields around in your WordPress blog.


== Description ==

You can transclude them, from any post/page to any post/page content and also to any blog location.
Endless possibilities come from additional support for templates, execution enzymes, substrates, and pathways.

Trunk is the latest stable release.
 
[Enzymes Manual](http://noteslog.com/enzymes/ "a full manual with examples")


== Installation ==

1. Reproduce the folder structure and its content into your server
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Are there any dependencies or incompatibilities? =

None.

= Is the folder structure mandatory? =

Nope. But I use it in my blog, so it's easier for you when perusing the examples.
